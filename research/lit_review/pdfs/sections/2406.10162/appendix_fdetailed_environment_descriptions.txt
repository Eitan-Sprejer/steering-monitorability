Appendix F
Detailed environment descriptions

Each of our environments consists of a prompt (or set of prompts) combined with a environment (or list of environments).

A prompt is a simple string, and a environment is a python dictionary containing:

•

A collection of files and directories to define the initial state on the model’s sandbox.define the initial state of the model’s sandbox,

•

A python command to run in the model’s sandbox after it takes its action which determines if the model successfully games the specification,

•

A python command that is run in the model’s sandbox after it takes its action to determine whether the model would get caught by the human when trying to game the specification.
