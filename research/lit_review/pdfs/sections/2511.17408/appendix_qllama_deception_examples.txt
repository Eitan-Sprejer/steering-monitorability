Q.1
Deception (trading)

This is the only dataset for which we use the same prompt throughout, as well as the only one relying on multi-turn model responses.
Some model responses are fixed while others are generated and then filtered for (like the model making the insider trade). We use an autograder LLM to label the example as deceptive if the model’s responses indicate it is hiding the use of insider knowledge/ “merger news”, instead of labelling based on whether the model makes the trade.
