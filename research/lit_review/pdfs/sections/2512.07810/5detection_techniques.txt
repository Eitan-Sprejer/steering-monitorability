5 Detection Techniques Without knowing which strategies the red team were using, the blue team developed a number of approaches: • Black Box Transcript Analysis. These included manually reading transcripts, using tools like Docent to summarise transcripts and debug trajectories ( meng2025docent ) , and using LLM agents to read and compare evaluation trajectories to find discrepancies between the untrusted model and the weaker trusted model. • Black Box Interventional Strategies. These included prompting / jailbreaking models for incrimination or performance elicitation, asking follow-up questions, and sampling tokens from user personas. • White Box Strategies. These included activation probes for deception. After seeing the results of the first phase, we decided to let the blue team have access to what we believed to be a more powerful class of elicitation strategies (but not obviously better detection strategies): • Training-based Strategies. These included on-distribution, cross-distribution, and anti-refusal training for performance elicitation, and attempting to leverage elicitation into detection of sandbagging. Unblinded analysis was also performed on a number of other detection techniques after the game, in order to determine if they could have been useful. These included noising activations, probing directly for sandbagging, and Sparse Auto Encoders (SAEs). Techniques which were pursued during the game were also extended and improved during un-blinded analysis, to determine if they could have been more useful.
