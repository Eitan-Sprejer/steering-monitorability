2 Preliminaries We study VLM-based evaluation of web agents (Deng et al. , 2023 ; Zheng et al. , 2024a ) , where the agent cannot manipulate the environment, allowing us to isolate failures arising from reasoning manipulation. We adopt a standard VLM-as-a-Judge framework (Xue et al. , 2025 ; Lù et al. , 2025 ) . An agent trajectory consists of K K steps: T = ( ( o i , c i , a i ) ) i = 1 K T=\bigl((o_{i},c_{i},a_{i})\bigr)_{i=1}^{K} , where o i o_{i} is the observation (e.g., screenshot), c i c_{i} the agent’s chain-of-thought, and a i a_{i} the action. A judge J J takes the full trajectory and outputs J ​ ( T ) ∈ { success , failure } J(T)\in\{\textsc{success},\textsc{failure}\} . Current evaluation assumes the agent’s CoT faithfully reflects its interaction with the environment. We focus on faithfulness with respect to (1) the environment state and (2) the user’s task—CoTs that contradict observations, fabricate progress, or misrepresent the task are unfaithful. We show this evaluation can be systematically manipulated: modifying only { c i } \{c_{i}\} while holding { o i } \{o_{i}\} and { a i } \{a_{i}\} fixed induces a large number of false positives.
